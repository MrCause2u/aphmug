# Aloysius Prophetic and Healing Ministries Website

A modern, responsive website for Aloysius Prophetic and Healing Ministries (APHM), also known as God's Embassy Church, located in Kyenjojo District, Uganda.

## 🌟 Features

### Design & User Experience
- **Modern, responsive design** that works perfectly on desktop, tablet, and mobile devices
- **Beautiful hero section** with blurred background effects and gradient overlays
- **Smooth animations** and scroll effects for enhanced user engagement
- **Accessible design** with proper focus states and ARIA labels
- **Fast loading** with optimized assets and performance considerations

### Content Sections
- **Hero Section**: Welcoming visitors with ministry overview
- **About Section**: History, mission, and community statistics
- **Ministry Focus**: Six key areas of ministry work
- **Children's Bible Study**: Information about the 2024 initiative
- **Donation Section**: Clear calls-to-action for supporting the ministry
- **Contact Section**: Complete contact information and social media links

### SEO Optimization
- **Comprehensive meta tags** for search engines
- **Structured data (JSON-LD)** for church organization
- **Open Graph tags** for social media sharing
- **Twitter Card meta tags** for Twitter sharing
- **Semantic HTML** with proper heading hierarchy
- **Optimized keywords** for Christian ministry, Uganda church, healing ministry
- **Fast loading speeds** for better search rankings

### Interactive Features
- **Smooth scrolling navigation** with active link highlighting
- **Mobile-responsive hamburger menu**
- **Back-to-top button** for easy navigation
- **Scroll-triggered animations** using Intersection Observer
- **Parallax effects** on hero background
- **Counter animations** for statistics
- **Form validation** ready for future contact forms

## 🚀 Getting Started

### Simple Setup
1. Download all files to your web server directory
2. Ensure all files are in the same folder:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
3. Open `index.html` in your web browser

### File Structure
```
/
├── index.html          # Main HTML file
├── styles.css          # All CSS styling
├── script.js           # JavaScript functionality
└── README.md           # This documentation
```

## 🔧 Customization

### Updating Content
1. **Ministry Information**: Edit the HTML content in `index.html`
2. **Contact Details**: Update phone numbers, email, and address in the contact section
3. **Social Media Links**: Modify URLs in the social media section
4. **Statistics**: Update numbers in the about section

### Styling Changes
1. **Colors**: Modify CSS custom properties in `styles.css`
2. **Fonts**: Change Google Fonts imports in the HTML head
3. **Layout**: Adjust grid layouts and spacing in CSS

### Adding Features
1. **Contact Form**: Add form HTML and enable form validation in `script.js`
2. **Image Gallery**: Add image elements and enable lazy loading
3. **Blog Section**: Extend the HTML structure for blog posts
4. **Events Calendar**: Add calendar functionality with external libraries
